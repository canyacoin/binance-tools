Binance Chain Tools

Tools for the Binance Chain ecosystem.

The CanYa Team will be building a suite of open-source tools that will enhance the capabilities of Binance Chain. Some of these tools will be integrated into the CanYa Project as well as other decentralised applications. The CanYa Team will be regularly updating this page with completed tools in the coming months.

MULTI-SEND
Multi-Sender Tool
Easily send transactions to multiple addresses.

This tool uses the Binance Chain batched transaction feature to send assets to multiple addresses easily. Users can manually add addresses, memos and amounts, or alternatively upload from a CSV file to send large amounts of transactions. Users pay the Batch Transaction fee.

MULTI-SIG
Multi-Signature Tool
Easily create and send from Binance Chain multi-signature accounts.

This tool uses Binance Chain multi-key store to persist state about multi-signature wallets on-chain and is non-interactive since signers donâ€™t need to be online at the same time. Users first set up the wallet by specifying the threshold and maximum number of signatures (such as a 2 of 3 wallet), and then sign in turn to send transactions from the wallet.

ESCROW
Escrow Tool
Easily escrow funds between two parties.

This tool is an adapted multi-signature module that places restrictions on how funds can be spent between parties. Users set up details about how the payment should be made, and once both parties are happy, funds are released to the correct recipient. The third-party can step in to process disputes and charges a small escrow fee.

HEDGED ESCROW
Hedged Escrow
Easily escrow funds between parties that is hedged to the price of an external asset.

This tool is an adapted escrow module that allows a float to be maintained, and an external price reference to be added (such as paying in BNB priced in USD). Users specify payment amounts in the price of the external asset such that price volatility risk is removed whilst the asset is escrowed. The float in the escrow underwrites payments and as such is very useful for marketplaces.

DAO
DAO Tool
Easily set up staking, voting and election dApps.

This tool allows developers to add staking, election and voting logic to their dApps. Users can stake assets on their wallets with a designated un-bonding period. They can then create elections that require minimum stake participation with designated outcomes. Votes can be tallied uses 1p1v or more elaborate schemes such as quadratic voting to determine an outcome.

CONTINUOUS LIQUIDITY POOLS
Liquidity Pools
On-chain continuous liquidity pools.

This module allows users to stake assets and BNB in on-chain pools, and then perform trustless swaps across pools. The module features always-on liquidity and fair market-based prices that are resistant to manipulation. Stakers earn fees when staking assets, and users can instantly swap assets in a single transaction.
